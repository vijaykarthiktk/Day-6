HTML Basics

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It structures the content and tells the browser how to display text, images, and other elements.

Essential Elements

  • <!DOCTYPE html> – Defines the document type.
  • <html> – The root of the HTML document.
  • <head> – Contains metadata like title, charset, etc.
  • <body> – Contains the visible page content.
  • <h1> to <h6> – Header tags, from largest (h1) to smallest (h6).
  • <p> – Paragraph element.
  • <a> – Anchor tag for hyperlinks.
  • <img> – Image element.
  • <div> and <span> – For layout and inline grouping.

Learning HTML is the first step to becoming a web developer. It's easy to get started and fun to build with!
